# Linear Design System

## Brand
Precision engineering tools. Dense, information-rich UI. No unnecessary decoration.

## Colors
- Background: #0F0F11 (near-black)
- Surface: #1A1A1F
- Border: #2A2A30
- Text primary: #F2F2F2
- Text secondary: #8A8A9A
- Accent: #5C66DB (indigo/violet)
- Accent hover: #6E78F0
- Success: #26A882
- Warning: #E6A93A
- Error: #E5534B

## Typography
- Font: "Inter", system-ui, -apple-system, sans-serif
- Heading: 600 weight, tight tracking (-0.02em)
- Body: 400 weight, 14px base, 1.5 line-height
- Mono: "JetBrains Mono", monospace

## Spacing
- Base unit: 4px
- Common: 4, 8, 12, 16, 20, 24, 32, 40, 48
- Border radius: 6px standard, 4px compact, 8px card

## Component Patterns
- Buttons: flat, tight padding (6px 12px), no border-radius > 6px
- Cards: subtle border + background lift, no shadow
- Lists: divider lines, no cards
- Input: borderless with bottom line or minimal border
- Icons: 16px default, Lucide-style stroke icons

## Voice
Efficient. Technical. No marketing fluff in UI copy. Action-first labels.
