# Apple Design System

## Brand
Premium consumer technology. Minimal. Material-aware. Deeply spatial in iOS/macOS, clean on web.

## Colors
- Background: #FFFFFF / #000000 (pure white or pure black)
- Surface: #F5F5F7 (light) / #1C1C1E (dark)
- Text primary: #1D1D1F (light) / #F5F5F7 (dark)
- Text secondary: #6E6E73
- Accent: #0071E3 (Apple blue)
- Red: #FF3B30
- Green: #34C759
- Border: rgba(0,0,0,0.1)

## Typography
- Font: "SF Pro Display", "SF Pro Text", -apple-system, BlinkMacSystemFont, sans-serif
- Heading large: 700 weight, -0.015em tracking, large scale (40px+)
- Heading section: 600 weight, 28–32px
- Body: 400 weight, 17px, 1.47 line-height
- Legal/caption: 400, 12px, #6E6E73

## Spacing
- Base unit: 8px
- Generous whitespace — padding is never tight
- Common: 8, 16, 20, 24, 32, 44, 56, 80, 96

## Component Patterns
- Buttons: pill-shaped or rectangle with 8px+ radius, never flat unstyled
- Cards: no hard borders — shadow or backdrop-blur for elevation
- Navigation: sticky, transparent with blur effect (backdrop-filter)
- Images: full-bleed, high-quality, product-photography aesthetic
- Sections: generous padding, centered content, max-width ~980px

## Voice
Aspirational. Human. Focus on the experience, not the feature. "Incredibly" is acceptable here.
