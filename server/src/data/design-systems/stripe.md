# Stripe Design System

## Brand
Enterprise-grade financial infrastructure. Trust + clarity. Clean, light, and precise.

## Colors
- Background: #FFFFFF
- Surface: #F6F9FC
- Border: #E3E8EF
- Text primary: #0A2540
- Text secondary: #425466
- Accent: #635BFF (Stripe purple)
- Accent dark: #4F46E5
- Success: #1EA672
- Info: #0080FF
- Error: #DF1B41

## Typography
- Font: "Sohne", "ui-sans-serif", -apple-system, sans-serif; fallback to Inter
- Heading: 600–700 weight, generous size scaling
- Body: 400 weight, 16px base, 1.6 line-height
- Label: 500 weight, 13px, 0.03em tracking (uppercase for metadata)

## Spacing
- Base unit: 8px
- Common: 4, 8, 16, 24, 32, 48, 64, 96
- Border radius: 4px buttons, 8px cards, 12px modals

## Component Patterns
- Buttons: solid fill, gentle shadow, 6px radius
- Cards: white bg, 1px border, 4px radius, subtle drop shadow
- Tables: alternating row subtle, header with weight
- Input: 1px border, focus ring in brand blue
- Gradient elements: used sparingly on hero/marketing sections only

## Voice
Clear. Confident. Trustworthy. Favors concrete over abstract. Short sentences.
