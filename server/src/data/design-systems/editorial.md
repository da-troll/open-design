# Editorial Monocle Design System

## Brand
High-end editorial. Magazine-quality layouts. Think Monocle, The Economist online, Substack Premium. Contrast, hierarchy, typographic excellence.

## Colors
- Background: #FDFDF9 (newsprint white)
- Surface: #F5F4EE
- Text primary: #1A1814
- Text secondary: #5C5850
- Rule/border: #CCCAB8
- Accent: #B5452A (editorial red — think masthead)
- Pull quote: #3A3128
- Caption: #8A8274

## Typography
- Headline: "Playfair Display", "Georgia", serif — 700–900 weight, generous size
- Subhead: "EB Garamond", serif — 600 weight
- Body: "EB Garamond" or "Merriweather" serif, 17–18px, 1.7 line-height (very readable)
- UI/caption: "Inter", sans-serif, 12–13px, 0.04em tracking uppercase
- Byline: italic serif, slightly smaller than body

## Spacing
- Wide margins: never let content go edge-to-edge
- Column grid: two or three column layouts preferred
- Common: 8, 16, 24, 32, 48, 64, 80
- Section breaks: horizontal rules (1px, full-width)
- Border radius: minimal — 0 or 2px max

## Component Patterns
- Navigation: horizontal, uppercase labels, thin underline on hover
- Hero: full-width image with text overlay or adjacent text column
- Pull quotes: enlarged, italic, left border rule
- Cards: image-first, 0px radius, thin border
- Buttons: all-caps, letter-spaced, flat or outline only

## Voice
Authoritative. Considered. No buzzwords. Long-form friendly. Slightly formal.
