# Skill: deck

## Output
A presentation deck (5–8 slides) rendered as a single scrollable HTML document.
Each slide is a 1280×720px frame (16:9) displayed full-width, stacked vertically.

## Structure
- Slide 1: Title + subtitle + visual
- Slide 2–3: Problem / context
- Slide 4–5: Solution / key points (one big idea per slide)
- Slide 6: Data/proof (chart placeholder or key metric)
- Slide 7–8: CTA / next steps
- Slide number bottom-right

## Content
Generate real, contextually relevant copy for the brief — not placeholder text.
Charts: render as CSS/SVG bar or line charts with actual data structure (made-up but plausible).

## Image placeholders
Use `<img-gen prompt="...">` for hero images and visual accents.

## Quality bar
Must look like a professional pitch deck — not a template. Apply design system to typography, color, and layout.
