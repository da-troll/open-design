# Skill: mobile-app

## Output
A mobile app screen set (3–5 key screens) rendered in a single HTML document.
Each screen is a 390×844px frame (iPhone 15 Pro dimensions) displayed side-by-side horizontally.

## Structure
- Status bar at top (12:00 / signal/battery icons)
- App content in middle
- Home indicator at bottom
- Navigation bar or tab bar where appropriate
- All screens coherent with each other (same app, same design language)
- Label each screen: Onboarding / Home / Detail / Settings / etc.

## Image placeholders
Use `<img-gen prompt="...">` for avatar images, product thumbnails, hero illustrations.

## Quality bar
Must look like a real app mockup, not a wireframe. Pixel-accurate to common mobile patterns.
